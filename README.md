# PrimerListManager

遺伝子工学などのDNAを扱う研究者の多くが使用している[ApE] (https://jorgensen.biology.utah.edu/wayned/ape/) が受け付けるPrimer Listの形式に瞬時に変換するソフトです。
このソフトウェアを開くだけで、更新してくれます。また、Fowerd primerとReverse primerの色もカラーパレットで変更することができます
